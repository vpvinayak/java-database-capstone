# Smart Clinic User Stories

## Doctor User Stories
- As a doctor, I can log in securely so that I can access my dashboard.
- As a doctor, I can view patient appointments so that I can manage consultations.
- As a doctor, I can access patient medical records so that I can prescribe accurate treatment.

## Patient User Stories
- As a patient, I can register and log in so that I can access clinic services.
- As a patient, I can search doctors by specialty and availability so that I can book appointments.
- As a patient, I can view my appointments so that I can track upcoming visits.

## Admin User Stories
- As an admin, I can add and manage doctors so that clinic information stays updated.
- As an admin, I can manage users and appointments so that the clinic system operates efficiently.
- As an admin, I can monitor system records so that clinic operations remain organized.
